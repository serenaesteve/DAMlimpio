1.-Tenéis que daros de alta en GitHub (sign in) 2.-Debéis crear uno o varios repositorios (o un repositorio con carpetas, o varios repositorios) 3.-Descargáis GitHub Desktop en vuestra máquina 4.-Clonáis el repositorio creado (File-Clone Repository) 5.-En explorador de Windows, ponéis todo vuestro código en la carpeta clonada 6.-Una vez que hayáis copiado los archivos, hacéis commit + push

Con eso habéis subido vuestros archivos a GitHub

7.-En github.com, 8.-Entráis en el repositorio 9.-Vais a settings (final de la barra superior) 10.-Entráis en collaborators 11-Manage access - add people - añadís invitación para el usuario "jocarsa"
